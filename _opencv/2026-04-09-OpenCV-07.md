Part 1: Image Enhancement | 图像增强

1.1 Gamma Correction | Gamma校正

人眼看亮度是非线性的（对暗部更敏感），而摄像头是线性的。Gamma校正用于补偿这种差异。

```python
import cv2
import numpy as np
import matplotlib.pyplot as plt

def cv_show(name, img):
    cv2.imshow(name, img)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

img = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test1.jpg')

# Gamma校正
gamma = 0.5  # gamma>1变暗，gamma<1变亮
lut = np.zeros(256, dtype=np.uint8)
for i in range(256):
    lut[i] = int(((i / 255.0) ** gamma) * 255)

gamma_result = cv2.LUT(img, lut)
res = np.hstack((gamma_result, img))
cv_show('res', res)
```

原理： 通过查表(LUT)对每个像素进行幂运算，gamma<1时暗部被提亮。

gamma值 效果 应用场景
< 1 暗部提亮 曝光不足的图像
= 1 不变 原始图像
1 整体变暗 过曝的图像

1.2 White Balance | 白平衡

白平衡用于纠正不同光源下的偏色，让白色物体看起来是白色。

```python
# 方法1：最大值法（假设最亮的地方是白色）
rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)

max_r = np.max(rgb[:, :, 0])
max_g = np.max(rgb[:, :, 1])
max_b = np.max(rgb[:, :, 2])

gain_r = max_g / max_r
gain_b = max_g / max_b

rgb[:, :, 0] = rgb[:, :, 0] * gain_r
rgb[:, :, 2] = rgb[:, :, 2] * gain_b
rgb = np.clip(rgb, 0, 255).astype(np.uint8)

wb_result = cv2.cvtColor(rgb, cv2.COLOR_RGB2BGR)
res = np.hstack((wb_result, img))
cv_show('res', res)

# 方法2：手动选区域法（框出白色区域）
# cv2.imshow('image', rgb)
# cv2.waitKey(0)
# white_region = rgb[100:300, 200:400]  # 手动指定白色区域
# ref_r = white_region[:,:,0].mean()
# ref_g = white_region[:,:,1].mean()
# ref_b = white_region[:,:,2].mean()
# gain_r = 255 / ref_r
# gain_g = 255 / ref_g
# gain_b = 255 / ref_b
# rgb[:,:,0] = rgb[:,:,0] * gain_r
# rgb[:,:,1] = rgb[:,:,1] * gain_g
# rgb[:,:,2] = rgb[:,:,2] * gain_b
```

白平衡方法选择指南：

场景 推荐方法 原理
有明显的白色物体 手动选区域法 告诉程序"这个区域应该是白色"
没有白色，场景颜色丰富 最大值法 假设最亮的地方是白色
整体偏一种颜色 灰度世界法 假设整张图平均是灰色
肤色要准（拍人） 不做白平衡 保留原色更自然
YOLO检测用 最大值法 检测对颜色相对不敏感

result_stretch = (img_float - min_val) * 255.0 / (max_val - min_val)

Part 2: Color Spaces | 颜色空间

2.1 HSV Color Space | HSV颜色空间

HSV按颜色过滤物体非常方便：H（色调）决定是什么颜色，S（饱和度）决定颜色浓淡，V（明度）决定亮度。

```python
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

# 查看各通道
h_channel = hsv[:, :, 0]  # 色调
s_channel = hsv[:, :, 1]  # 饱和度
v_channel = hsv[:, :, 2]  # 明度
cv_show('h_channel', h_channel)
cv_show('s_channel', s_channel)
cv_show('v_channel', v_channel)

# 提取红色区域（H在0-10或170-180）
lower_red = np.array([0, 50, 50])
upper_red = np.array([10, 255, 255])
mask = cv2.inRange(hsv, lower_red, upper_red)
red_only = cv2.bitwise_and(img, img, mask=mask)
cv_show('red_only', red_only)
```

H值 颜色 H值 颜色
0 红色 60 绿色
30 橙色 120 蓝色
45 黄色 150 紫色

2.2 YUV Color Space | YUV颜色空间

YUV把颜色拆成Y（亮度）和UV（色度）。人眼对亮度更敏感，可压缩UV通道节省带宽。

```python
yuv = cv2.cvtColor(img, cv2.COLOR_BGR2YUV)
y_channel = yuv[:, :, 0]   # 亮度（灰度图）
u_channel = yuv[:, :, 1]   # 蓝色色度
v_channel = yuv[:, :, 2]   # 红色色度
cv_show('y_channel', y_channel)

back_to_bgr = cv2.cvtColor(yuv, cv2.COLOR_YUV2BGR)
cv_show('back_to_bgr', back_to_bgr)
```

YUV在ISP中的三种核心应用：

```python
# 1. UV分辨率减半（420格式）- 节省带宽
h, w = yuv.shape[:2]
y = yuv[:, :, 0]
u = yuv[::2, ::2, 1]  # 每隔一行一列取一个U
v = yuv[::2, ::2, 2]  # 每隔一行一列取一个V

# 2. 只在Y通道做降噪（不影响颜色）
yuv = cv2.cvtColor(img, cv2.COLOR_BGR2YUV)
y_denoised = cv2.GaussianBlur(yuv[:, :, 0], (5, 5), 1)
yuv[:, :, 0] = y_denoised
result = cv2.cvtColor(yuv, cv2.COLOR_YUV2BGR)

# 3. 颜色调整（单独调U或V）- 让图像偏蓝
yuv[:, :, 1] = np.clip(yuv[:, :, 1] + 20, 0, 255)  # U通道+20
result = cv2.cvtColor(yuv, cv2.COLOR_YUV2BGR)
```

工业界ISP芯片设计流程：

```
RAW图 → 去马赛克 → RGB → YUV
                           ↓
                    Y通道：降噪、锐化、对比度增强
                    UV通道：降采样、颜色校正
                           ↓
                    输出YUV（给编码器或显示）
```

Part 3: Geometric Transformations | 几何变换

3.1 Affine Transformation | 仿射变换

仿射变换保持直线的平行性（平移、旋转、缩放、剪切）。

```python
img = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test8.jpg')
h, w = img.shape[:2]

# 原图的三个点（左上、右上、左下）
src = np.float32([[50, 50], [w-50, 50], [50, h-50]])
# 三个点要移动到的位置
dst = np.float32([[80, 80], [w-20, 80], [80, h-20]])

M = cv2.getAffineTransform(src, dst)   # 计算2x3变换矩阵
result_aff = cv2.warpAffine(img, M, (w, h))
cv_show('result_aff', result_aff)
```

3.2 Perspective Transformation | 透视变换（重点）

透视变换用于将斜着拍的物体变成正视角，是商标检测、车牌识别、文档扫描的核心技术。

```python
# 手动标出歪的物体的4个角（用画图工具先看坐标）
src = np.float32([
    [100, 300],   # 左上
    [500, 280],   # 右上
    [520, 480],   # 右下
    [80, 500]     # 左下
])

# 计算物体的宽度和高度
width = int(max(np.linalg.norm(src[1] - src[0]), np.linalg.norm(src[2] - src[3])))
height = int(max(np.linalg.norm(src[3] - src[0]), np.linalg.norm(src[2] - src[1])))

# 目标：正矩形
dst = np.float32([
    [0, 0],
    [width-1, 0],
    [width-1, height-1],
    [0, height-1]
])

M = cv2.getPerspectiveTransform(src, dst)
result_pers = cv2.warpPerspective(img, M, (width, height))
cv_show('result_pers', result_pers)
```

变换类型 自由度 保持性质 应用
仿射变换 6 平行性 图像校正、旋转
透视变换 8 直线（不保持平行） 文档扫描、商标矫正

3.3 Image Registration | 图像配准

把不同角度拍摄的同一场景对齐到同一坐标系，用于拼接或对比。

```python
img1 = cv2.imread('scene1.jpg', 0)
img2 = cv2.imread('scene2.jpg', 0)

# 创建ORB特征检测器
orb = cv2.ORB_create()

# 找特征点
kp1, des1 = orb.detectAndCompute(img1, None)
kp2, des2 = orb.detectAndCompute(img2, None)

# 匹配特征点
bf = cv2.BFMatcher(cv2.NORM_HAMMING, crossCheck=True)
matches = bf.match(des1, des2)
matches = sorted(matches, key=lambda x: x.distance)[:30]  # 取前30个最好的

# 提取匹配点坐标
src_pts = np.float32([kp1[m.queryIdx].pt for m in matches]).reshape(-1, 1, 2)
dst_pts = np.float32([kp2[m.trainIdx].pt for m in matches]).reshape(-1, 1, 2)

# 计算变换矩阵（RANSAC去除错误匹配）
M, _ = cv2.findHomography(src_pts, dst_pts, cv2.RANSAC, 5.0)

# 应用变换
aligned = cv2.warpPerspective(img2, M, (img1.shape[1], img1.shape[0]))
```

💡 Some Problems I Met... | 遇到的问题

Problem 1: Gamma校正后图像过暗或过亮

· What's wrong? 使用gamma=0.5时图像变得很亮，gamma=2时变得很暗。
· Reason gamma<1时暗部被提亮，gamma>1时亮部被压暗，这是数学性质决定的。
· Solution 根据图像实际情况选择gamma值。曝光不足用gamma<1（如0.5），过曝用gamma>1（如1.5-2.0）。

```python
# 动态调整gamma
def adjust_gamma(image, gamma=1.0):
    inv_gamma = 1.0 / gamma
    table = np.array([(i / 255.0) ** inv_gamma * 255 for i in range(256)]).astype(np.uint8)
    return cv2.LUT(image, table)
```

---

Problem 2: 白平衡后图像偏色更严重

· What's wrong? 应用白平衡后图像反而偏红或偏蓝。
· Reason 最大值法假设最亮点是白色，但如果场景中最亮点本身就是彩色（如红色灯笼），增益计算就会错误。
· Solution 改用灰度世界法，或手动框出真正的白色区域。

```python
# 灰度世界法
avg_r = np.mean(rgb[:, :, 0])
avg_g = np.mean(rgb[:, :, 1])
avg_b = np.mean(rgb[:, :, 2])
avg_all = (avg_r + avg_g + avg_b) / 3
gain_r = avg_all / avg_r
gain_g = avg_all / avg_g
gain_b = avg_all / avg_b
```

---

Problem 3: 透视变换后图像模糊

· What's wrong? 矫正后的图像分辨率低，看起来很模糊。
· Reason 透视变换会对原图像素进行重采样，如果目标尺寸(width, height)设置太小，会丢失细节。
· Solution 适当增大目标尺寸，或使用cv2.INTER_CUBIC插值。

```python
result_pers = cv2.warpPerspective(img, M, (width*2, height*2), flags=cv2.INTER_CUBIC)
```

---

Problem 4: ORB特征匹配效果差

· What's wrong? 匹配的特征点很少，或者有很多错误匹配。
· Reason 两张图像差异太大（角度、光照、缩放），或者场景纹理不明显。
· Solution 使用更鲁棒的特征（SIFT），或调整RANSAC参数。

```python
# 使用SIFT（需要cv2.xfeatures2d或opencv-contrib）
# sift = cv2.SIFT_create()
# kp1, des1 = sift.detectAndCompute(img1, None)
# 使用FLANN匹配器
FLANN_INDEX_KDTREE = 1
index_params = dict(algorithm=FLANN_INDEX_KDTREE, trees=5)
search_params = dict(checks=50)
flann = cv2.FlannBasedMatcher(index_params, search_params)
matches = flann.knnMatch(des1, des2, k=2)
```

```python
def gamma_correction(img, gamma=1.0):
    """Gamma校正（查表法，适合FPGA）"""
    lut = np.array([((i / 255.0) ** (1.0/gamma)) * 255 for i in range(256)]).astype(np.uint8)
    return cv2.LUT(img, lut)

def white_balance_max_rgb(img):
    """最大值法白平衡"""
    rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
    max_r, max_g, max_b = np.max(rgb, axis=(0, 1))
    gain_r = max_g / max_r
    gain_b = max_g / max_b
    rgb[:, :, 0] = np.clip(rgb[:, :, 0] * gain_r, 0, 255)
    rgb[:, :, 2] = np.clip(rgb[:, :, 2] * gain_b, 0, 255)
    return cv2.cvtColor(rgb.astype(np.uint8), cv2.COLOR_RGB2BGR)

def perspective_correction(img, src_points):
    """透视变换矫正（输入4个角点）"""
    src = np.float32(src_points)
    width = int(max(np.linalg.norm(src[1] - src[0]), np.linalg.norm(src[2] - src[3])))
    height = int(max(np.linalg.norm(src[3] - src[0]), np.linalg.norm(src[2] - src[1])))
    dst = np.float32([[0, 0], [width-1, 0], [width-1, height-1], [0, height-1]])
    M = cv2.getPerspectiveTransform(src, dst)
    return cv2.warpPerspective(img, M, (width, height))

def extract_color_mask(img, color='red'):
    """HSV颜色过滤"""
    hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)
    color_ranges = {
        'red': [([0, 50, 50], [10, 255, 255]), ([170, 50, 50], [180, 255, 255])],
        'green': [([40, 50, 50], [80, 255, 255])],
        'blue': [([100, 50, 50], [130, 255, 255])],
    }
    masks = []
    for lower, upper in color_ranges[color]:
        mask = cv2.inRange(hsv, np.array(lower), np.array(upper))
        masks.append(mask)
    return cv2.bitwise_or(masks[0], masks[1]) if len(masks) > 1 else masks[0]
```
