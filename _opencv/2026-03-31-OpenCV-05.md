## 📖 What have I learned? | 学习内容

### Part 1: The Steps of Canny | Canny五步流程

Canny边缘检测通过五个阶段将原始图像转化为清晰的边缘图：

#### Step 1: Gaussian Filter | 高斯滤波

首先使用高斯滤波器平滑图像，消除噪声干扰。

```python
# 高斯核示例（σ=1.4，3x3）
H = np.array([
    [0.0924, 0.1192, 0.0924],
    [0.1192, 0.1538, 0.1192],
    [0.0924, 0.1192, 0.0924]
])
# 卷积计算：e = H * A（逐元素相乘后求和）
```

**为什么需要这一步？** 边缘检测对噪声非常敏感，高斯滤波可以有效抑制噪声，避免将噪点误检为边缘。

#### Step 2: Gradient Calculation | 梯度计算

使用Sobel算子计算每个像素点的梯度强度和方向。（前面的Learning Notes已经有所涉猎）

```python
# Sobel算子
Sx = np.array([[-1, 0, 1],
               [-2, 0, 2],
               [-1, 0, 1]])  # 水平方向

Sy = np.array([[1, 2, 1],
               [0, 0, 0],
               [-1, -2, -1]])  # 垂直方向

# 梯度强度 G = sqrt(Gx² + Gy²)
# 梯度方向 θ = arctan(Gy / Gx)
```

**梯度方向的意义：** 边缘方向垂直于梯度方向，为下一步的非极大值抑制提供依据。

#### Step 3: Non-Maximum Suppression | 非极大值抑制（NMS） Important!

在梯度方向上，只保留局部最大值，抑制非边缘的杂散响应。

**原理：** 对于每个像素，沿着梯度方向比较前后两个像素的梯度值，若当前像素不是最大值，则置为0。

**简化方法：** 将梯度方向离散为8个方向（0°, 45°, 90°, 135°等），只需比较相邻两个方向上的像素即可，无需复杂的插值计算。

```
梯度方向离散化示例：
    135°  90°  45°
      \   |   /
       \  |  /
    180°——●——0°
       /  |  \
      /   |   \
    225° 270° 315°
```

#### Step 4: Double Threshold | 双阈值检测

使用高阈值和低阈值将边缘分为三类：

```python
# 双阈值逻辑
if gradient > maxVal:
    edge = strong    # 强边缘，保留
elif gradient > minVal:
    edge = weak      # 弱边缘，需进一步判断
else:
    edge = discard   # 非边缘，舍弃
```

**弱边缘处理规则：** 若弱边缘与强边缘相连，则保留为真实边缘；否则抑制。

#### Step 5: Edge Tracking | 边缘跟踪

通过连通性分析，抑制孤立的弱边缘，完成最终边缘检测。

### Part 2: Canny in OpenCV | OpenCV中的Canny实现 Pretty convenient!

OpenCV提供了简洁的`cv2.Canny()` API，封装了上述所有步骤。

```python
import cv2
import matplotlib.pyplot as plt
import numpy as np

def cv_show(name, img):
    cv2.imshow(name, img)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

# 读取灰度图像
img = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test1.jpg', cv2.IMREAD_GRAYSCALE)

# Canny边缘检测
v1 = cv2.Canny(img, 80, 150)   # 高阈值150，低阈值80
v2 = cv2.Canny(img, 50, 100)   # 高阈值100，低阈值50

# 可视化对比
plt.figure(figsize=(8, 4))
plt.subplot(121)
plt.imshow(v1, cmap='gray')
plt.axis('off')
plt.title('Large threshold (80, 150)')

plt.subplot(122)
plt.imshow(v2, cmap='gray')
plt.axis('off')
plt.title('Small threshold (50, 100)')

plt.tight_layout()
plt.show()

print('可以看出，minVal和maxVal越小，边缘检测越细致丰富，但也更容易混入噪声')
```

### Part 3: Threshold Selection | 阈值选择的影响

```python
# 更多阈值组合的实验 More tests
img = cv2.imread("car.png", cv2.IMREAD_GRAYSCALE)

thresholds = [(120, 250), (80, 150), (50, 100)]
results = []

for minVal, maxVal in thresholds:
    edge = cv2.Canny(img, minVal, maxVal)
    results.append(edge)

# 对比不同阈值的效果 Comparisons
res = np.hstack(results)
cv_show(res, 'res')
```

**阈值选择规律：**

| 阈值组合 | 效果 | 适用场景 |
|----------|------|----------|
| **高阈值较大** | 边缘稀疏，只保留最强边缘 | 轮廓清晰、噪声少的图像 |
| **低阈值较小** | 边缘丰富，细节更多 | 细节丰富的图像 |
| **两者都小** | 边缘最细致，但易混入噪声 | 需要先滤波再检测 |

## 💡 Some Problems I Met... | 遇到的问题

### Problem 1: Understanding Non-Maximum Suppression | 理解非极大值抑制

**What's wrong? ：** 刚开始不理解为什么梯度计算后还要做非极大值抑制。

**Reason：** Sobel等算子计算出的边缘通常较粗（多个像素宽），非极大值抑制能将边缘"瘦身"到单像素宽度。

**Solution：** 理解算法本质——只保留梯度方向上的局部最大值，消除边缘附近的渐变响应。

---

### Problem 2: Threshold Selection Confusion | 阈值选择困惑

**What's wrong? ：** 不知道如何选择合适的minVal和maxVal。

**Reason：** Canny的双阈值没有固定公式，需要根据图像特性和应用场景调整。

**Solution：**
- 经验比例：低阈值 = 高阈值 × 0.3 ~ 0.5
- 先用较大阈值获取主要边缘，再逐步降低观察细节变化
- 对噪声较多的图像，适当提高两个阈值

---

### Problem 3: Gaussian Blur vs Canny Built-in Smoothing | 高斯模糊与Canny内置平滑

**What's wrong? ：** 是否需要手动先做高斯滤波再调用Canny？

**Reason：** `cv2.Canny()`内部已经包含了高斯滤波步骤，无需手动预处理。

**Solution：** 直接使用`cv2.Canny()`即可。如果需要更强的平滑效果，可以在调用前手动滤波。

```python
# 标准用法
edges = cv2.Canny(img, 50, 100)

# 手动加强平滑
blurred = cv2.GaussianBlur(img, (5, 5), 0)
edges = cv2.Canny(blurred, 50, 100)
```

## 📚 CODE Repository | 代码仓库

```python
def canny_edge_detection(img, low_threshold=50, high_threshold=100, 
                          apply_gaussian=False, gaussian_kernel=(5,5)):
    """
    Canny边缘检测封装函数
    
    Parameters:
    - img: 输入图像（灰度图）
    - low_threshold: 低阈值
    - high_threshold: 高阈值
    - apply_gaussian: 是否额外应用高斯滤波
    - gaussian_kernel: 高斯核大小
    """
    if apply_gaussian:
        img = cv2.GaussianBlur(img, gaussian_kernel, 0)
    
    edges = cv2.Canny(img, low_threshold, high_threshold)
    return edges

def compare_thresholds(img, thresholds_list):
    """对比不同阈值组合的Canny结果"""
    results = []
    for low, high in thresholds_list:
        edges = cv2.Canny(img, low, high)
        results.append(edges)
    
    # 水平拼接显示
    comparison = np.hstack(results)
    return comparison
```
