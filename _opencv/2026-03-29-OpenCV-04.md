### PART1：形态学基础

形态学操作通常要求**白色前景、黑色背景**。如果图像相反，可以用`cv2.bitwise_not()`反转。

```python
import cv2
import matplotlib.pyplot as plt
import numpy as np

def cv_show(name, img):
    cv2.imshow(name, img)
    cv2.waitKey(0)
    cv2.destroyAllWindows()
```

#### 1.1 腐蚀与膨胀

腐蚀和膨胀是形态学操作的基础，通过结构元素（kernel）在图像上滑动实现。

```python
# 创建测试图像：200x200黑色背景，中心有一个白色矩形和黑色圆形
img = np.zeros((200, 200), dtype=np.uint8)
cv2.rectangle(img, (50, 50), (150, 150), 255, -1)  # 白色矩形
cv2.circle(img, (100, 100), 30, 0, -1)             # 黑色圆形（洞）

# 反转图像（白色前景重要）
img_inv = cv2.bitwise_not(img)

# 定义结构元素（卷积核）
kernel = np.ones((5, 5), np.uint8)  # 核越大，效果越强

# 腐蚀：消除白色区域边界
erosion = cv2.erode(img_inv, kernel, iterations=1)
erosion_true = cv2.bitwise_not(erosion)  # 反转回原图风格

# 膨胀：扩大白色区域
dilation = cv2.dilate(img_inv, kernel, iterations=1)

# 可视化
plt.figure(figsize=(12, 4))
plt.subplot(131)
plt.imshow(img_inv, 'gray')
plt.title('Original (Inverted)')
plt.axis('off')

plt.subplot(132)
plt.imshow(erosion, 'gray')
plt.title('Erosion')
plt.axis('off')

plt.subplot(133)
plt.imshow(dilation, 'gray')
plt.title('Dilation')
plt.axis('off')

plt.tight_layout()
plt.show()
```

**腐蚀与膨胀的原理 principles!：**

| 操作 | 公式 | 效果 | 应用 |
|------|------|------|------|
| **腐蚀** | 核区域内全为1才保留1 | 缩小白色区域，消除细小突出 | 去除噪点、分离粘连物体 |
| **膨胀** | 核区域内有一个1就置1 | 扩大白色区域，填充细小空洞 | 连接断裂区域、填补空洞 |

**参数说明：**
- `iterations`：迭代次数，次数越多效果越强
- `kernel`：结构元素大小，越大效果越明显

#### 1.2 开运算与闭运算

```python
# 创建测试图像：一个圆形
img = np.zeros((200, 200), dtype=np.uint8)
cv2.circle(img, (100, 100), 60, 255, -1)

kernel = np.ones((5, 5), np.uint8)

# 开运算：先腐蚀再膨胀，消除小噪点或毛刺
opening = cv2.morphologyEx(img, cv2.MORPH_OPEN, kernel)

# 闭运算：先膨胀再腐蚀，填补黑色小洞
closing = cv2.morphologyEx(img, cv2.MORPH_CLOSE, kernel)
```

**开运算与闭运算对比：**

| 操作 | 执行顺序 | 效果 | 典型应用 |
|------|----------|------|----------|
| **开运算** | 腐蚀 → 膨胀 | 消除小物体、毛刺 | 去噪、分离物体 |
| **闭运算** | 膨胀 → 腐蚀 | 填补小洞、连接断裂 | 填充空洞、连接邻近物体 |

---

#### 1.3 梯度运算

梯度 = 膨胀 - 腐蚀，提取物体的边界轮廓。

```python
# Method 1：手动计算
dilation = cv2.dilate(img, kernel, iterations=1)
erosion = cv2.erode(img, kernel, iterations=1)
manual_gradient = dilation - erosion

# Method 2：使用API Recommend!
gradient = cv2.morphologyEx(img, cv2.MORPH_GRADIENT, kernel)
```

#### 1.4 礼帽与黑帽

```python
# 读取真实图像
img = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test1.jpg')
kernel = np.ones((5, 5), np.uint8)

# 礼帽 = 原图 - 开运算（提取噪点和小毛刺）
tophat = cv2.morphologyEx(img, cv2.MORPH_TOPHAT, kernel)

# 黑帽 = 闭运算 - 原图（提取未被填充的黑洞）
blackhat = cv2.morphologyEx(img, cv2.MORPH_BLACKHAT, kernel)
```

**四种高级形态学操作总结：**

| 操作 | 公式 | 提取内容 |
|------|------|----------|
| **开运算** | 腐蚀→膨胀 | 去噪后的主体 |
| **闭运算** | 膨胀→腐蚀 | 填补空洞后的主体 |
| **梯度** | 膨胀 - 腐蚀 | 物体边界 |
| **礼帽** | 原图 - 开运算 | 噪点、毛刺 |
| **黑帽** | 闭运算 - 原图 | 空洞、裂缝 |

### PART2：边缘检测算子

边缘检测是图像中灰度发生剧烈变化的区域，为Canny等高级边缘检测算法打基础。

#### 2.1 Sobel算子

Sobel算子通过计算像素点在水平方向和垂直方向的梯度来检测边缘。

```python
# 创建测试图像：一个白色圆形
img = np.zeros((200, 200), dtype=np.uint8)
cv2.circle(img, (100, 100), 60, 255, -1)

# 水平方向梯度（检测垂直边缘）
sobelx = cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=3)
sobelx = cv2.convertScaleAbs(sobelx)  # 取绝对值，避免负数截断

# 垂直方向梯度（检测水平边缘）
sobely = cv2.Sobel(img, cv2.CV_64F, 0, 1, ksize=3)
sobely = cv2.convertScaleAbs(sobely)

# 合并两个方向
sobelxy = cv2.addWeighted(sobelx, 0.5, sobely, 0.5, 0)
```

**Sobel算子原理 principles!：**
- 水平方向：检测垂直边缘（左右明暗变化）
- 垂直方向：检测水平边缘（上下明暗变化）
- 合并后得到完整的边缘信息

#### 2.2 Scharr算子

Scharr算子是Sobel的改进版，对边缘的检测更加精细。

```python
# Scharr算子（没有ksize参数）
scharrx = cv2.Scharr(img, cv2.CV_64F, 1, 0)
scharrx = cv2.convertScaleAbs(scharrx)

scharry = cv2.Scharr(img, cv2.CV_64F, 0, 1)
scharry = cv2.convertScaleAbs(scharry)

scharrxy = cv2.addWeighted(scharrx, 0.5, scharry, 0.5, 0)
```

#### 2.3 Laplacian算子

Laplacian算子同时检测所有方向的边缘，但对noise敏感。

```python
# Laplacian算子（无dx、dy参数，无ksize参数）
laplacian = cv2.Laplacian(img, cv2.CV_64F)
laplacian = cv2.convertScaleAbs(laplacian)
```

**三种算子对比：**

| 算子 | 参数特点 | 效果 | 注意事项 |
|------|----------|------|----------|
| **Sobel** | 有ksize参数 | 基础边缘检测 | 需取绝对值 |
| **Scharr** | 无ksize参数 | 更精细的边缘 | 比Sobel更敏感 |
| **Laplacian** | 无dx、dy参数 | 所有方向同时检测 | 对noise敏感，需先滤波 |

#### 2.4 可视化对比

```python

plt.subplot(131)
plt.imshow(sobelxy, 'gray')
plt.title('Sobel')
plt.axis('off')

plt.subplot(132)

plt.title('Scharr')
plt.axis('off')

plt.subplot(133)

```

## 💡 Some problems I met... Ah

### Problem 1：Sobel算子输出只有半边轮廓

**What's wrong? ：** 使用Sobel算子后，只检测到一半的边缘（如只有左半圆或右半圆）。

**Reason：** 梯度计算时，从黑到白的变化为正，从白到黑的变化为负。`cv2.Sobel()`默认输出包含负数，而显示时负数被截断为0。我一开始忘了这一点。

**Solution：** 使用`cv2.convertScaleAbs()`取绝对值。

```python
# ❌ 错误做法 WRONG!
sobelx = cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=3)
cv_show('sobelx', sobelx)  # 只显示一半

# ✅ 正确做法
sobelx = cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=3)
sobelx = cv2.convertScaleAbs(sobelx)
cv_show('sobelx', sobelx)  # 显示完整边缘
```

### Problem 2：形态学操作效果相反

**What's wrong? ：** 腐蚀操作后，白色区域反而变大了。

**Reason：** 形态学操作默认处理**白色前景、黑色背景**。如果图像是黑色前景、白色背景，效果会完全相反。

**Solution：** 使用`cv2.bitwise_not()`反转图像。

```python
# 图像反转
img_inv = cv2.bitwise_not(img)

# 对反转后的图像进行形态学操作
erosion = cv2.erode(img_inv, kernel)

# 如果需要，再反转回来
result = cv2.bitwise_not(erosion)
```

### Problem 3：Scharr算子报错

**What's wrong? ：** `cv2.Scharr(img, cv2.CV_64F, 1, 0, ksize=3)`报错。

**Reason：** Scharr算子没有`ksize`参数，它的核大小是固定的3x3。

**Solution：** 移除ksize参数。

```python
# ❌ 错误 WRONG!
scharrx = cv2.Scharr(img, cv2.CV_64F, 1, 0, ksize=3)

# ✅ 正确
scharrx = cv2.Scharr(img, cv2.CV_64F, 1, 0)
```

### Problem 4：Laplacian算子检测边缘不清晰

**What's wrong? ：** Laplacian检测出的边缘很杂乱，有很多噪点。

**Reason：** Laplacian对噪声非常敏感，直接对原图使用会放大噪声。

**Solution：** 先进行高斯滤波（LoG：Laplacian of Gaussian）。

```python
# 先模糊再检测
blurred = cv2.GaussianBlur(img, (3, 3), 0)
laplacian = cv2.Laplacian(blurred, cv2.CV_64F)
laplacian = cv2.convertScaleAbs(laplacian)
```

## 💡 学习心得

### 关键知识点总结 Summary

1. **形态学操作的核心是结构元素**：kernel大小和形状直接影响操作效果
2. **前景背景规则**：牢记白色前景、黑色背景，用bitwise_not快速反转
3. **梯度计算要取绝对值**：避免负数被截断导致边缘丢失
4. **算子选择策略**：
   - 基础需求用Sobel
   - 精细边缘用Scharr
   - 需要各向同性检测用Laplacian（记得先滤波）

本阶段学习的核心函数封装：

```python
def morph_operation(img, operation='open', kernel_size=5, iterations=1):
    """统一的形态学操作接口"""
    kernel = np.ones((kernel_size, kernel_size), np.uint8)
    
    ops = {
        'erode': cv2.MORPH_ERODE,
        'dilate': cv2.MORPH_DILATE,
        'open': cv2.MORPH_OPEN,
        'close': cv2.MORPH_CLOSE,
        'gradient': cv2.MORPH_GRADIENT,
        'tophat': cv2.MORPH_TOPHAT,
        'blackhat': cv2.MORPH_BLACKHAT
    }
    
    if operation in ops:
        return cv2.morphologyEx(img, ops[operation], kernel, iterations=iterations)
    else:
        raise ValueError(f"Unknown operation: {operation}")

def edge_detection(img, method='sobel', blur_first=False):
    """统一的边缘检测接口"""
    if blur_first:
        img = cv2.GaussianBlur(img, (3, 3), 0)
    
    if method == 'sobel':
        sobelx = cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=3)
        sobely = cv2.Sobel(img, cv2.CV_64F, 0, 1, ksize=3)
        sobelx = cv2.convertScaleAbs(sobelx)
        sobely = cv2.convertScaleAbs(sobely)
        return cv2.addWeighted(sobelx, 0.5, sobely, 0.5, 0)
    
    elif method == 'scharr':
        scharrx = cv2.Scharr(img, cv2.CV_64F, 1, 0)
        scharry = cv2.Scharr(img, cv2.CV_64F, 0, 1)
        scharrx = cv2.convertScaleAbs(scharrx)
        scharry = cv2.convertScaleAbs(scharry)
        return cv2.addWeighted(scharrx, 0.5, scharry, 0.5, 0)
    
    elif method == 'laplacian':
        laplacian = cv2.Laplacian(img, cv2.CV_64F)
        return cv2.convertScaleAbs(laplacian)
    
    else:
        raise ValueError(f"Unknown method: {method}")
```
