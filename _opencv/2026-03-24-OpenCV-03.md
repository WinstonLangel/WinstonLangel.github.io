### Part1：图像数值计算与融合

#### 1.1 像素运算基础

OpenCV中像素运算有两种方式，行为完全不同：

```python
import cv2
import matplotlib.pyplot as plt
import numpy as np

# 读取图像
img1 = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test1.jpg')
```

##### 直接使用 `+` 运算符（取模运算）

```python
# 每个像素值加10
img1_1 = img1 + 10

# 观察特定区域的像素变化
print(img1[70:74, :, 0])      # 原始像素值
print(img1_1[70:74, :, 0])    # 加10后的像素值

# 两个图像相加（越界时取模）
print((img1 + img1_1)[70:74, :, 0])  # 越界数据 % 256
```

**Key points：**
- `img + 10` 会对每个像素值加10
- 两个图像用`+`相加时，**超过255的值会取模256**
- 例如：250 + 10 = 260 % 256 = 4（溢出）

##### 使用 `cv2.add()` 函数（截断运算）

```python
print(cv2.add(img1, img1_1)[70:74, :, 0])  # 越界全部按255处理
```

**两种方式对比：**

| 操作 | `+` 运算符 | `cv2.add()` |
|------|-----------|-------------|
| 越界处理 | 取模（%256） | 截断（max=255） |
| 效果 | 可能产生伪影 | 符合预期 |
| 推荐场景 | 特殊效果 | **日常使用** |

#### 1.2 图像尺寸统一

图像融合前必须保证两张图片尺寸相同。（这点很重要，我一开始没注意到，导致报错）

```python
# 读取两张图片
img_cat = cv2.imread('cat.jpg')
img_dog = cv2.imread('dog.jpg')

print(f"猫图尺寸: {img_cat.shape}")   # (414, 500, 3)
print(f"狗图尺寸: {img_dog.shape}")   # (600, 800, 3)
```

##### 三种尺寸统一方法

```python
# Method A：将狗图缩放到猫图大小
h_cat, w_cat = img_cat.shape[:2]
img_dog_resized = cv2.resize(img_dog, (w_cat, h_cat))

# Method B：将猫图缩放到狗图大小
h_dog, w_dog = img_dog.shape[:2]
img_cat_resized = cv2.resize(img_cat, (w_dog, h_dog))

# Method C：统一缩放到中间尺寸（Highly recommend!）
target_size = (640, 480)
img_cat_resized = cv2.resize(img_cat, target_size)
img_dog_resized = cv2.resize(img_dog, target_size)

# 检查形状是否一致
print(img_cat_resized.shape)   # (480, 640, 3)
print(img_dog_resized.shape)   # (480, 640, 3)
```

**NOTES：**
- `cv2.resize()` 的参数顺序是 **(width, height)**，不是 (height, width)
- `img.shape` 返回的是 (height, width, channels)

#### 1.3 图像加权融合

`cv2.addWeighted()` 是实现图像融合的核心函数，可以实现透明混合效果。

```python
# 加权融合：result = img1 * alpha + img2 * beta + gamma
res = cv2.addWeighted(img1, 0.4, img2, 0.6, 0)
```

**参数说明：**
- `alpha`：第一张图片的权重（0-1）
- `beta`：第二张图片的权重（0-1）
- `gamma`：亮度调节值，0表示不添加额外亮度

##### 完整融合demo

```python
import cv2
import matplotlib.pyplot as plt

# 1. 读取两张图片
img_cat = cv2.imread('cat.jpg')
img_dog = cv2.imread('dog.jpg')

# 2. 统一尺寸（以猫图为标准）
h_cat, w_cat = img_cat.shape[:2]
img_dog_resized = cv2.resize(img_dog, (w_cat, h_cat))

# 3. 图像融合
result = cv2.addWeighted(img_cat, 0.6, img_dog_resized, 0.4, 0)

# 4. 可视化对比
plt.figure(figsize=(15, 5))

plt.subplot(131)
plt.imshow(cv2.cvtColor(img_cat, cv2.COLOR_BGR2RGB))
plt.title('Cat')
plt.axis('off')

plt.subplot(132)
plt.imshow(cv2.cvtColor(img_dog_resized, cv2.COLOR_BGR2RGB))
plt.title('Dog (resized)')
plt.axis('off')

plt.subplot(133)
plt.imshow(cv2.cvtColor(result, cv2.COLOR_BGR2RGB))
plt.title('Blended (Cat 0.6, Dog 0.4)')
plt.axis('off')

plt.tight_layout()
plt.show()
```

### Part2：图像阈值分割

阈值分割是最简单的图像分割方法，通过设定阈值将图像分为前景和背景。

```python
# 读取图像并转换色彩空间
img = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test1.jpg')
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
img_gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 六种阈值化方法
ret1, thresh1 = cv2.threshold(img_gray, 127, 255, cv2.THRESH_BINARY)
print(f'BINARY使用的阈值：{ret1}')

ret2, thresh2 = cv2.threshold(img_gray, 127, 255, cv2.THRESH_BINARY_INV)
ret3, thresh3 = cv2.threshold(img_gray, 127, 255, cv2.THRESH_TRUNC)
ret4, thresh4 = cv2.threshold(img_gray, 127, 255, cv2.THRESH_TOZERO)
ret5, thresh5 = cv2.threshold(img_gray, 127, 255, cv2.THRESH_TOZERO_INV)

# OTSU自适应阈值（自动计算最优阈值）
ret6, thresh6 = cv2.threshold(img_gray, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
print(f"OTSU自动计算的阈值：{ret6}")

# 可视化对比
titles = ['ORIGINAL', 'ORIGINAL_GRAY', 'BINARY', 'BINARY_INV', 
          'TRUNC', 'TOZERO', 'TOZERO_INV', 'BINARY+OTSU']
images = [img_rgb, img_gray, thresh1, thresh2, thresh3, thresh4, thresh5, thresh6]

for i in range(8):
    plt.subplot(2, 4, i+1), plt.imshow(images[i], 'gray')
    plt.title(titles[i])
    plt.xticks([]), plt.yticks([])
plt.tight_layout()  # 避免重叠
plt.show()
```

#### 六种阈值化方法详解

| 方法 | 参数 | 原理 | 应用场景 |
|------|------|------|----------|
| **BINARY** | `THRESH_BINARY` | >127设为255，≤127设为0 | 最简单二值化，对比度高的图像 |
| **BINARY_INV** | `THRESH_BINARY_INV` | BINARY的反转 | 需要黑白颠倒的场景 |
| **TRUNC** | `THRESH_TRUNC` | >127设为127，其余不变 | 抑制高亮区域 |
| **TOZERO** | `THRESH_TOZERO` | >127不变，≤127设为0 | 保留高亮部分，去掉暗区 |
| **TOZERO_INV** | `THRESH_TOZERO_INV` | TOZERO的反转 | 保留暗部，去掉高亮 |
| **OTSU** | `THRESH_BINARY + THRESH_OTSU` | 自动计算最优阈值 | 双峰直方图图像，最常用 |

### 第三部分：噪声类型与图像平滑滤波

#### 3.1 噪声类型深度解析

##### 椒盐噪声 (Salt-and-Pepper Noise)

**视觉表现：** 图像中随机出现的黑白点，像撒了盐和胡椒粉一样

**产生原因：**
- 传感器故障
- 传输信道干扰
- 内存位元错误

**最佳滤波器：** **中值滤波**

##### 高斯噪声 (Gaussian Noise)

**视觉表现：** 图像整体呈现细密的颗粒感，像老电视的雪花

**产生原因：**
- 传感器暗光环境下的热噪声
- 电子电路的自然波动

**最佳滤波器：** **高斯滤波**

#### 3.2 四种平滑滤波器

##### 均值滤波 (Average Blur)

```python
blur = cv2.blur(img, (3, 3))  # 卷积核3x3
cv_show('blur', blur)
```

**特点：** 速度快，但会模糊边缘

##### 方框滤波 (Box Filter)

```python
box = cv2.boxFilter(img, -1, (5, 5), normalize=True)
cv_show('box', box)
```

**特点：** 均值滤波的变种，可选择是否归一化

##### 高斯滤波 (Gaussian Blur)

```python
gaussian = cv2.GaussianBlur(img, (3, 3), 0)  # 标准差0表示自动计算
cv_show('gaussian', gaussian)
```

**特点：** 保留边缘效果最好，去高斯噪声最优

##### 中值滤波 (Median Blur)

```python
median = cv2.medianBlur(img, 3)  # 3表示3x3邻域
cv_show('median', median)
```

**特点：** 去椒盐噪声最优，保留边缘清晰

#### 3.3 四种滤波器效果对比

```python
# 横向拼接四种滤波结果
res = np.hstack((blur, box, gaussian, median))

plt.figure(figsize=(16, 10))
plt.imshow(cv2.cvtColor(res, cv2.COLOR_BGR2RGB))
plt.title('Comparisons with different blurs')
plt.xticks([]), plt.yticks([])
plt.show()
```

#### 滤波器选择指南

| 场景 | 推荐 | 原因 |
|------|------|------|
| **快速去噪** | 均值滤波 | 计算速度最快 |
| **保留边缘** | 高斯滤波 | 权重分布合理，边缘过渡自然 |
| **去除椒盐噪声** | 中值滤波 | 有效消除孤立黑白点 |
| **去除高斯噪声** | 高斯滤波 | 噪声分布与滤波器匹配 |
| **视频实时处理** | 均值滤波 | 速度优先 |
| **照片精细处理** | 高斯滤波 | 效果最佳 |

## 💡 学习心得

### 关键知识点总结

1. **`+` vs `cv2.add()`**：图像处理中优先使用`cv2.add()`，避免取模运算产生的伪影

2. **尺寸统一陷阱**：`cv2.resize()`参数是(width, height)，而`img.shape`返回(height, width)

3. **OTSU自动阈值**：在实际项目中非常实用，避免了手动调参的麻烦

4. **噪声类型决定滤波选择**：
   - 孤立黑白点 → 中值滤波
   - 均匀颗粒感 → 高斯滤波

### 对项目的启发

**YOLOv8室内定位系统：**
- **数据增强**：用图像融合模拟不同光照条件
- **预处理**：先用高斯滤波去噪，再用OTSU辅助特征提取
- **边缘保护**：定位需要保留边缘信息，高斯滤波比均值滤波更适合

**SuperAgent智能体：**
- **多传感器融合**：融合摄像头和红外图像
- **实时处理**：视频流处理时优先考虑均值滤波的速度优势

## 📚 代码仓库

本阶段学习的核心函数封装：

```python
def safe_add(img1, img2):
    """安全的图像加法（饱和截断）"""
    return cv2.add(img1, img2)

def blend_images(img1, img2, alpha=0.5, beta=0.5, gamma=0):
    """图像加权融合（自动统一尺寸）"""
    h, w = img1.shape[:2]
    img2_resized = cv2.resize(img2, (w, h))
    return cv2.addWeighted(img1, alpha, img2_resized, beta, gamma)

def apply_thresholds(img_gray):
    """应用所有阈值化方法"""
    thresholds = {}
    thresholds['binary'], thresh1 = cv2.threshold(img_gray, 127, 255, cv2.THRESH_BINARY)
    thresholds['otsu'], thresh_otsu = cv2.threshold(img_gray, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
    return thresholds, thresh_otsu

def denoise_image(img, method='gaussian', kernel_size=3):
    """统一的去噪接口"""
    if method == 'mean':
        return cv2.blur(img, (kernel_size, kernel_size))
    elif method == 'gaussian':
        return cv2.GaussianBlur(img, (kernel_size, kernel_size), 0)
    elif method == 'median':
        return cv2.medianBlur(img, kernel_size)
    elif method == 'box':
        return cv2.boxFilter(img, -1, (kernel_size, kernel_size), normalize=True)
```
