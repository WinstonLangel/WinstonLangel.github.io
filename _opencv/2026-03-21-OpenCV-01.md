## 📖 What have I learned? 学习内容

### 1. 图像操作 IMAGE

#### 1.1 图像读取与显示

```python
import cv2
import matplotlib.pyplot as plt
import numpy as np

# 读取图像（BGR格式）
img = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test1.jpg')

# 自定义显示函数
def cv_show(name, img):
    cv2.imshow(name, img)
    cv2.waitKey(0)      # 等待按键quit
    cv2.destroyAllWindows()

cv_show('image', img)
```

**关键点：**
- `cv2.imread()` 默认读取为BGR格式，与常规的RGB不同
- `cv2.waitKey(0)` 表示无限等待按键，常用于调试
- 图像显示后必须配合销毁窗口，否则会造成内存占用

#### 1.2 图像属性探究

```python
# 查看图像形状 (height, width, channels)
print(img.shape)

# 灰度图读取
img_gray = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test1.jpg', cv2.IMREAD_GRAYSCALE)
cv_show('gray', img_gray)

# 查看数据类型和大小
print(type(img_gray))   # <class 'numpy.ndarray'>
print(img_gray.size)    # 像素点总数
print(img_gray.dtype)   # uint8
```

**收获：**
- 彩色图像shape为 (H, W, 3) dim = 3，灰度图为 (H, W) dim = 2
- OpenCV读取的图像直接转为numpy数组，方便后续数学运算
- 灰度图每个像素0-255，uint8类型，节省内存空间

### 2. 视频处理 VIDEO

```python
# 读取视频文件
vc = cv2.VideoCapture(r"C:\Users\user李响\Desktop\英语配音\哈利波特.mp4")

if vc.isOpened():
    open, frame = vc.read()
else:
    open = False

while open:
    ret, frame = vc.read()
    if frame is None:
        break
    if ret == True:
        # 转为灰度图显示
        gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
        cv2.imshow('result', gray)
        
        # 按ESC退出
        if cv2.waitKey(30) & 0xFF == 27:
            break

vc.release()
cv2.destroyAllWindows()
```

**技术要点：**
- `cv2.VideoCapture()` 可以读取视频文件或摄像头（参数为0/1）
- `cvtColor()` 进行色彩空间转换，BGR→GRAY是常用操作
- `waitKey(30)` 控制播放速度，参数为延迟毫秒数（处理完这一帧等待多少毫秒）
- 释放资源是良好习惯，避免内存泄漏

## 💡 学习心得

### Some problems I met

1. **路径问题**：Windows下路径我使用了原始字符串 `r'` 或双反斜杠 `\\`，避免转义字符错误

2. **BGR vs RGB**：一开始没注意这个差异，用matplotlib显示时颜色异常，后来才意识到OpenCV默认BGR

3. **视频帧读取**：循环读取时一定要判断是否 `frame is None`，否则视频结束时可能报错
