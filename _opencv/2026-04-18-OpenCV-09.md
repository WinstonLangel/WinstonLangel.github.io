### Part 1: Meanshift Target Tracking | Meanshift目标跟踪  

#### 1.1 原理——用“颜色气味”追踪目标  
Meanshift的核心思路非常直觉：  
1. 在**第一帧**手动框出目标，计算框内HSV颜色直方图（这就是目标的“气味”）。  
2. 对**后续每一帧**，先用`calcBackProject`算出整张图里哪些区域与这个“气味”相似（反向投影图）。  
3. 然后Meanshift算法在这张概率图上迭代移动窗口，直到“爬”到颜色密度最高的位置——那就是目标新位置。  

**项目应用场景**：商场里跟踪某个穿红衣服的顾客（人流分析），或者跟踪一个红色Logo在画面中的移动轨迹。由于只依赖颜色，对目标形变不敏感，但对光照变化比较脆弱——这就是为什么前置ISP处理很重要。

#### 1.2 代码一步步拆解  

##### Step 1：打开视频，手动框选目标  
```python
cap = cv2.VideoCapture(r'C:\Users\user李响\Desktop\英语配音\哈利波特.mp4')
ret, frame = cap.read()

bbox = cv2.selectROI('框出跟踪目标', frame, fromCenter=False, showCrosshair=True)
x, y, w, h = bbox
```

`selectROI`弹出一个交互窗口，用鼠标拖拽框选。返回值是 (x, y, w, h)。`showCrosshair=True` 显示十字准星，更方便精准框选。

##### Step 2：计算目标的颜色直方图（“气味”）

```python
roi = frame[y:y+h, x:x+w]
hsv_roi = cv2.cvtColor(roi, cv2.COLOR_BGR2HSV)
hist = cv2.calcHist([hsv_roi], [0], None, [180], [0, 180])
cv2.normalize(hist, hist, 0, 255, cv2.NORM_MINMAX)
```

- 注意HSV的H通道范围是0~180（OpenCV将360度压缩了一半）。只用了H（色调）通道，因为颜色主要由色调决定，忽略明度V可以减少光照干扰。

##### Step 3：逐帧跟踪

```python
while True:
    ret, frame = cap.read()
    if not ret:
        break
    
    hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
    backproj = cv2.calcBackProject([hsv], [0], hist, [0, 180], 1)
    
    term_crit = (cv2.TERM_CRITERIA_EPS | cv2.TERM_CRITERIA_COUNT, 10, 1)
    ret, bbox = cv2.meanShift(backproj, (x, y, w, h), term_crit)
    
    x, y, w, h = bbox
    cv2.rectangle(frame, (x, y), (x+w, y+h), (0, 255, 0), 2)
    cv_show('final', frame)   # 别忘了之前这个函数的封装哈，因为老是写那一套流程太麻烦了。
```

- `calcBackProject`：生成反向投影图，每个像素值代表该位置颜色属于目标的概率。
- `term_crit`：迭代停止条件——最多移动10次，或中心移动距离小于1像素。
- **meanShift**：在概率图上迭代，返回新的边界框（大小不变）。

### Part 2: Camshift——自适应框大小与旋转

#### 2.1 Meanshift的痛点

Meanshift的框大小是固定的！如果目标走近变大，框只能覆盖一部分；如果目标旋转，框也歪不了。Camshift（Continuously Adaptive Meanshift）解决了这个问题。

#### 2.2 只需改一行代码

```python
ret, bbox = cv2.CamShift(backproj, (x, y, w, h), term_crit)
# bbox 返回的是旋转矩形 ((center_x, center_y), (width, height), angle)
```

Camshift会根据概率分布的二阶矩自动调整窗口大小和旋转角度。画框时需要转换一下：

```python
pts = cv2.boxPoints(ret)  # 获取旋转矩形的四个顶点
pts = np.int0(pts)
cv2.polylines(frame, [pts], True, (0, 255, 0), 2)
```

**项目应用场景**：商场里用户拿着手机走向Logo，画面中Logo逐渐变大——Camshift能**自适应**放大框，持续锁定目标。旋转不变性对于倾斜拍摄也有效。

### Part 3: 视频流ISP处理管线

#### 3.1 为什么需要视频ISP预处理？

YOLO是在理想图像上训练的，但真实视频流可能有：偏色、过暗、噪声。每一帧都做预处理，YOLO才能稳定发挥。

#### 3.2 完整管线代码

```python
cap = cv2.VideoCapture(r'C:\Users\user李响\Desktop\英语配音\哈利波特.mp4')
fps = cap.get(cv2.CAP_PROP_FPS)   # fps是帧数，我摄影的时候就了解过。听说最近大疆出了4K240帧？
width = int(cap.get(cv2.CAP_PROP_FRAME_WIDTH))
height = int(cap.get(cv2.CAP_PROP_FRAME_HEIGHT))

out = cv2.VideoWriter('output.mp4', cv2.VideoWriter_fourcc('m', 'p', '4', 'v'), fps, (width, height))

while True:
    ret, frame = cap.read()
    if not ret:
        break
    
    # ============ ISP处理（以Gamma为例） ============
    temp = frame.copy()
    cv2.intensity_transform.gammaCorrection(frame, temp, gamma=1.5)
    frame = temp
    # =============================================
    
    cv2.imshow('ISP processed', frame)
    out.write(frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
out.release()
cv2.destroyAllWindows()
```

#### 3.3 参数说明

- `cv2.VideoWriter_fourcc('m', 'p', '4', 'v')`：MP4编码格式。
- **fps、width、height**必须与原视频一致，否则输出文件损坏。
- `cv2.waitKey(1)`：控制播放速度，同时监听按键退出。

**项目应用场景**：

- **智视寻迹**：从视频流中实时提取帧，Gamma校正增强暗区Logo，白平衡统一色温，然后送YOLO检测。
- **SuperAgent**：摄像头画面实时ISP处理，保证输入视觉算法的图像质量稳定。

## 💡 Some Problems I Met... | 踩到的坑。。。

- Problem 1: Meanshift框不动或漂走
**Reason**：目标颜色与背景相似，反向投影图噪声大。
**Solution**：尽量选颜色独特的物体；可对反向投影图做形态学处理（开运算）去噪。

- Problem 2: Camshift返回的bbox不能直接画矩形
**Reason**：Camshift返回的是旋转矩形((cx,cy), (w,h), angle)。
**Solution**：用cv2.boxPoints()转成四个顶点，再用cv2.polylines()画。

Problem 3: VideoWriter输出文件打不开
**Reason**：编码格式不对或尺寸不匹配。
**Solution**：确保fps、width、height与原视频一致；编码用'm','p','4','v'通用性好。

Problem 4: waitKey(1)导致视频播放飞快
**Reason**：waitKey(1)只是延迟1ms，处理耗时如果很短，播放会加速。
**Solution**：可以计算处理时间，动态调整waitKey延迟，或者直接保存视频不看实时显示。

def track_with_camshift(video_path, save_path=None):
    """Camshift目标跟踪，可选保存处理后的视频"""
    cap = cv2.VideoCapture(video_path)
    ret, frame = cap.read()
    bbox = cv2.selectROI('Select Target', frame, False)
    x, y, w, h = bbox
    roi = frame[y:y+h, x:x+w]
    hsv_roi = cv2.cvtColor(roi, cv2.COLOR_BGR2HSV)
    hist = cv2.calcHist([hsv_roi], [0], None, [180], [0, 180])
    cv2.normalize(hist, hist, 0, 255, cv2.NORM_MINMAX)
    
    if save_path:
        fps = cap.get(cv2.CAP_PROP_FPS)
        w_frame = int(cap.get(cv2.CAP_PROP_FRAME_WIDTH))
        h_frame = int(cap.get(cv2.CAP_PROP_FRAME_HEIGHT))
        out = cv2.VideoWriter(save_path, cv2.VideoWriter_fourcc('m','p','4','v'), fps, (w_frame, h_frame))
    
    term_crit = (cv2.TERM_CRITERIA_EPS | cv2.TERM_CRITERIA_COUNT, 10, 1)
    
    while True:
        ret, frame = cap.read()
        if not ret:
            break
        hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
        backproj = cv2.calcBackProject([hsv], [0], hist, [0, 180], 1)
        ret, bbox = cv2.CamShift(backproj, (x, y, w, h), term_crit)
        pts = cv2.boxPoints(ret)
        pts = np.int0(pts)
        cv2.polylines(frame, [pts], True, (0, 255, 0), 2)
        
        if save_path:
            out.write(frame)
        cv2.imshow('Tracking', frame)
        if cv2.waitKey(1) & 0xFF == ord('q'):
            break
    
    cap.release()
    if save_path:
        out.release()
    cv2.destroyAllWindows()

def video_isp_pipeline(video_path, gamma=1.5, output_path=None):
    """视频流ISP预处理管线（Gamma校正示例）"""
    cap = cv2.VideoCapture(video_path)
    fps = cap.get(cv2.CAP_PROP_FPS)
    w = int(cap.get(cv2.CAP_PROP_FRAME_WIDTH))
    h = int(cap.get(cv2.CAP_PROP_FRAME_HEIGHT))
    
    if output_path:
        out = cv2.VideoWriter(output_path, cv2.VideoWriter_fourcc('m','p','4','v'), fps, (w, h))
    
    while True:
        ret, frame = cap.read()
        if not ret:
            break
        temp = frame.copy()
        cv2.intensity_transform.gammaCorrection(frame, temp, gamma=gamma)
        frame = temp
        
        if output_path:
            out.write(frame)
        cv2.imshow('ISP Output', frame)
        if cv2.waitKey(1) & 0xFF == ord('q'):
            break
    
    cap.release()
    if output_path:
        out.release()
    cv2.destroyAllWindows()
```
