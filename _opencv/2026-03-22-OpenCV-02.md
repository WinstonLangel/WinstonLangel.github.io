### 1. ROI区域截取

ROI是图像处理中最基础也最实用的技术，通过切片操作快速定位感兴趣区域。

```python
import cv2
import matplotlib.pyplot as plt
import numpy as np

# 读取图像
img = cv2.imread('cat.jpg')

# 定义显示函数
def cv_show(name, img):
    cv2.imshow(name, img)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

# ROI截取：获取前200行、前200列的区域
cat_roi = img[0:200, 0:200]
cv_show('cat_roi', cat_roi)
```

**Key points：**
- ROI截取本质是numpy数组的切片操作：`img[行起始:行结束, 列起始:列结束]`，其中结束并不包含
- 截取后的ROI是原图的**视图（view）**，修改ROI会影响原图
- 常用于目标检测的候选区域提取

**应用思考：**
- YOLOv8中，可以先通过ROI提取可能的检测区域，减少计算量
- 室内定位中，可以用ROI框定摄像头画面中的关键区域（如门、窗）

### 2. 颜色通道处理

OpenCV默认使用BGR色彩空间，理解三通道的独立操作是图像处理的基础。

#### 2.1 通道分离与合并

```python
# 分离BGR三个通道
b, g, r = cv2.split(img)

# 查看单通道的shape（二维数组）
print(b.shape)  # (414, 500)
print(r.shape)  # (414, 500)

# 合并通道还原彩色图像
img_merged = cv2.merge((b, g, r))
print(img_merged.shape)  # (414, 500, 3)
```

**Note!**
- `cv2.split()` 返回的是三个独立的二维数组
- 分离操作比较耗时，生产环境中通常用numpy索引直接操作
- 单通道图像是灰度信息，数值范围0-255

#### 2.2 单通道显示实验

```python
# 创建副本避免修改原图
cur_img = img.copy()

# 只保留R通道（将B和G通道置零）
cur_img[:, :, 0] = 0  # B通道置零
cur_img[:, :, 1] = 0  # G通道置零
cv_show('Only R Channel', cur_img)

# 只保留G通道
cur_img = img.copy()
cur_img[:, :, 0] = 0  # B通道置零
cur_img[:, :, 2] = 0  # R通道置零
cv_show('Only G Channel', cur_img)

# 只保留B通道
cur_img = img.copy()
cur_img[:, :, 1] = 0  # G通道置零
cur_img[:, :, 2] = 0  # R通道置零
cv_show('Only B Channel', cur_img)
```

**实验收获：**
- B通道：保留蓝色信息（天空、水面突出）
- G通道：保留绿色信息（草地、植物突出）
- R通道：保留红色信息（物体轮廓明显）

**numpy索引技巧 Some tricks：**
- `img[:, :, 0]` 表示所有行、所有列、第0通道（B）
- 修改指定通道可以实现色彩过滤效果

### 3. 边界填充（Padding）

边界填充在卷积神经网络中至关重要，直接影响特征图的尺寸和边缘信息的保留。

```python
# 定义填充尺寸（上下左右各50像素）
top_size, bottom_size, left_size, right_size = (50, 50, 50, 50)

# 五种填充方式
replicate = cv2.copyMakeBorder(img, top_size, bottom_size, left_size, right_size, 
                                borderType=cv2.BORDER_REPLICATE)

reflect = cv2.copyMakeBorder(img, top_size, bottom_size, left_size, right_size, 
                             borderType=cv2.BORDER_REFLECT)

reflect101 = cv2.copyMakeBorder(img, top_size, bottom_size, left_size, right_size, 
                                 borderType=cv2.BORDER_REFLECT_101)

wrap = cv2.copyMakeBorder(img, top_size, bottom_size, left_size, right_size, 
                          borderType=cv2.BORDER_WRAP)

constant = cv2.copyMakeBorder(img, top_size, bottom_size, left_size, right_size, 
                              borderType=cv2.BORDER_CONSTANT, value=0)

# 可视化对比
plt.figure(figsize=(12, 8))

plt.subplot(231), plt.imshow(cv2.cvtColor(img, cv2.COLOR_BGR2RGB)), plt.title('ORIGINAL')
plt.subplot(232), plt.imshow(cv2.cvtColor(replicate, cv2.COLOR_BGR2RGB)), plt.title('REPLICATE')
plt.subplot(233), plt.imshow(cv2.cvtColor(reflect, cv2.COLOR_BGR2RGB)), plt.title('REFLECT')
plt.subplot(234), plt.imshow(cv2.cvtColor(reflect101, cv2.COLOR_BGR2RGB)), plt.title('REFLECT_101')
plt.subplot(235), plt.imshow(cv2.cvtColor(wrap, cv2.COLOR_BGR2RGB)), plt.title('WRAP')
plt.subplot(236), plt.imshow(cv2.cvtColor(constant, cv2.COLOR_BGR2RGB)), plt.title('CONSTANT')

plt.tight_layout()
plt.show()
```

#### 五种填充方式详解 Details of the five methods

| 方式 | 参数 | 原理 | 应用场景 |
|------|------|------|----------|
| **REPLICATE** | `BORDER_REPLICATE` | 复制边缘像素 | 最简单的填充，保持边缘连续性 |
| **REFLECT** | `BORDER_REFLECT` | 镜像反射 | 常用于图像增强，过渡自然 |
| **REFLECT_101** | `BORDER_REFLECT_101` | 镜像反射 | 与REFLECT类似，平滑边缘处理 |
| **WRAP** | `BORDER_WRAP` | 对侧像素环绕 | 纹理生成、周期性图像处理 |
| **CONSTANT** | `BORDER_CONSTANT` | 固定值填充（常为0） | CNN卷积中的padding，最常用 |

**Something important：**
- 卷积神经网络中常用`BORDER_CONSTANT`，填充值为0（zero-padding）
- 填充可以控制卷积后特征图的尺寸变化
- 保留边缘信息，避免卷积核无法覆盖边界像素（目前不太理解，后续会继续深入学习）

```python
# 工具函数集合
def cv_show(name, img):
    """快速显示图像"""
    cv2.imshow(name, img)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

def extract_roi(img, x, y, w, h):
    """提取ROI区域"""
    return img[y:y+h, x:x+w]

def show_single_channel(img, channel):
    """显示单个通道"""
    cur = img.copy()
    if channel == 'R':
        cur[:, :, 0] = 0
        cur[:, :, 1] = 0
    elif channel == 'G':
        cur[:, :, 0] = 0
        cur[:, :, 2] = 0
    elif channel == 'B':
        cur[:, :, 1] = 0
        cur[:, :, 2] = 0
    cv_show(f'Only {channel} Channel', cur)
```
