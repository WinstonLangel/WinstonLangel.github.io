### Part 1: ORB Feature Detection & Matching | ORB特征检测与匹配  

#### 1.1 ORB是个啥？  
ORB（Oriented FAST and Rotated BRIEF）是OpenCV里最常用的特征检测算法之一。它的工作分两步：  
1. **找特征点（Keypoints）**：在图像里找角点、纹理丰富的地方。  
2. **生成描述子（Descriptors）**：给每个特征点发一张“身份证”（一串二进制编码），用来和其他图的特征点比对。  

说白了，ORB就是给图像里的关键位置做标记，然后比对两张图的标记是不是同一批——匹配上了就是同一个物体。SIFT和SURF虽然更准，但ORB免费开源且速度飞快，在嵌入式实时场景里是首选。

#### 1.2 代码实现  

```python
template = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test8.jpg', 0)   # 商标模板
scene = cv2.imread('scene.png', 0)   # YOLO裁剪出的候选区域

orb = cv2.ORB_create()
kp1, des1 = orb.detectAndCompute(template, None)
kp2, des2 = orb.detectAndCompute(scene, None)

bf = cv2.BFMatcher(cv2.NORM_HAMMING, crossCheck=True)
matches = bf.match(des1, des2)
matches = sorted(matches, key=lambda x: x.distance)

best_match_distance = matches[0].distance
print(f'最好的匹配距离为{best_match_distance}.')

if best_match_distance < 50:   # 小于30代表非常匹配，大于50代表不匹配
    print('是同一个商标！')
else:
    print('不是同一个商标！')
```

**关键参数解释**：

- `cv2.NORM_HAMMING`：ORB生成的是二进制描述子，用Hamming距离（不同bit的个数）衡量相似度。
- `crossCheck=True`：双向匹配，A匹配到B且B也匹配到A才算有效，过滤掉单向误匹配。（一开始忘了加这个，发现啥都能匹配上。。。）
- 距离阈值50：经验值，<30是高度确信，30~50可接受，>50基本不是同一物体。

**项目应用场景**：
在智视寻迹中，YOLO检测到候选框后，裁剪出来送ORB和数据库中的Logo模板比对。匹配距离小于阈值→确认店铺身份→完成视觉定位。这比单纯依赖YOLO的分类置信度更robust，尤其适合相似Logo的场景。

### Part 2: Image Quality Assessment | 图像质量评估（ISP调试核心）

这部分是**ISP工程师**（不好意思夹带私货了，平常爱搞点摄影）的必修课。调完参数跑个PSNR/SSIM，数值说话，拒绝玄学调参。

#### 2.1 PSNR（峰值信噪比）

**原理**：计算两张图像素级别的差异，值越大说明差异越小，画质损失越少。

```python
original = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test8.jpg')
processed = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test7.jpg')

psnr_value = cv2.PSNR(original, processed)
print(f'PSNR: {psnr_value:.2f}dB')

# 或者用skimage
from skimage.metrics import peak_signal_noise_ratio as psnr
psnr_value = psnr(original, processed, data_range=255)
print(f'PSNR: {psnr_value:.2f}dB')
```

**典型值参考**：

- 40dB：肉眼几乎看不出差异
- 30~40dB：轻微失真，可接受
- <30dB：明显失真，需调整算法参数

**项目应用场景**：
SuperAgent写降噪或锐化算法后，用PSNR对比处理前后图像，数值太低说明“下手太重”，得回调参数。

#### 2.2 SSIM（结构相似性）

**原理**：PSNR只看像素差，SSIM还综合亮度、对比度、结构，更符合人眼感受。值越接近1越像。

```python
# 还是那句，I'm a lazy boyyyy, so I'll just use the function directly...
from skimage.metrics import structural_similarity as ssim
ssim_value = ssim(original, processed, channel_axis=2)   # 彩色图需指定通道轴
print(f'SSIM: {ssim_value:.4f}')
```

**!!!Warning!!!踩坑提醒**：

- 图片太小会报错，先resize
- 读入彩色图必须加channel_axis=2，否则skimage默认当灰度图处理

这两个一定注意，我两个坑都踩了，无敌。

**项目应用场景**：
ISP调试gamma曲线时，对比不同参数的SSIM，选最接近1的那个参数组合。SSIM比PSNR更符合实际观感。

### Part 3: Noise Estimation | 噪声估计

**原理**：**Laplacian算子**提取高频信息（噪声+边缘），计算其标准差作为噪声水平估计。

```python
img = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test1.jpg')
laplacian = cv2.Laplacian(img, cv2.CV_64F)
noise_level = np.std(laplacian)
print(f'Noise level: {noise_level:.2f}')

if noise_level < 10:
    print('噪声很小，无需降噪')
elif noise_level < 30:
    print('噪声中等，建议降噪')
else:
    print('噪声很大，必须降噪')
```

**阈值参考（经验值）**：

- <10：噪声极小，画面干净
- 10~30：中等噪声
- 30：严重噪声，需强力降噪

### Part 4: Sharpness Detection | 清晰度检测

**原理**：用**Laplacian算子**（还是他！可见其重要性）提取边缘，计算方差作为清晰度指标。值越大边缘越丰富，图像越清晰。

```python
img = cv2.imread(r'D:\python-learn\python-YOLOv8\images\test1.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
laplacian = cv2.Laplacian(gray, cv2.CV_64F)
focus_measure = laplacian.var()

threshold = 1000   # 根据实际场景标定
if focus_measure < threshold:
    print(f'图片模糊，清晰度为: {focus_measure:.2f}')
else:
    print(f'图片清晰，清晰度为: {focus_measure:.2f}')
```

**为什么用Laplacian方差法？**

- 计算简单，适合硬件加速
- 对失焦和运动模糊敏感
- Tenengrad、Brenner等方法更复杂，Laplacian方差在嵌入式上最实用

**项目应用场景**：

- **智视寻迹**：用户拍照时实时检测清晰度，模糊则提示重拍，避免无效识别。
- **SuperAgent智能体应用开发**：机器人自主判断“此时可以拍照吗”，清晰度达标才触发识别模块，节省算力。
- **自动对焦(AF)**：清晰度值作为反馈信号，驱动镜头马达寻找峰值位置。（这个等我到实力强劲之时会实践）

## 💡 Some Problems I Met... | 踩到的坑。。。

### Problem 1: ORB匹配距离阈值设多少合适？
一开始设30，结果很多正确匹配被拒。后来统计了一批正负样本的匹配距离，发现正样本集中在20~45，负样本>60，最终定50为阈值。实际部署前最好针对自己的数据做标定。

### Problem 2: skimage的SSIM报错“image too small”
SSIM内部会做高斯平滑，图片尺寸小于7x7就报错。解决方法：cv2.resize()放大或跳过。

### Problem 3: 彩色图SSIM忘记加channel_axis
报错说数组维度不对，查文档才发现彩色图必须指明通道轴。加上channel_axis=2解决。

### Problem 4: 清晰度阈值不知道怎么定
跑了一批清晰和模糊样本，画了直方图，发现清晰图方差>1500，模糊图<800，取中间值1000做阈值。

## 📚 CODE Repository | 代码仓库

```python
def orb_match_score(template_path, scene_path, threshold=50):
    # ORB匹配，返回是否同一物体及最佳匹配距离
    template = cv2.imread(template_path, 0)
    scene = cv2.imread(scene_path, 0)
    orb = cv2.ORB_create()
    _, des1 = orb.detectAndCompute(template, None)
    _, des2 = orb.detectAndCompute(scene, None)
    bf = cv2.BFMatcher(cv2.NORM_HAMMING, crossCheck=True)
    matches = bf.match(des1, des2)
    if len(matches) == 0:
        return False, float('inf')
    best_dist = sorted(matches, key=lambda x: x.distance)[0].distance
    return best_dist < threshold, best_dist

def compute_psnr_ssim(img1, img2):
    # 计算PSNR和SSIM
    psnr_val = cv2.PSNR(img1, img2)
    from skimage.metrics import structural_similarity as ssim
    ssim_val = ssim(img1, img2, channel_axis=2)
    return psnr_val, ssim_val

def estimate_noise(img):
    # 噪声水平估计
    laplacian = cv2.Laplacian(img, cv2.CV_64F)
    return np.std(laplacian)

def sharpness_score(img):
    # 清晰度检测（Laplacian方差）
    gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
    return cv2.Laplacian(gray, cv2.CV_64F).var()
```
