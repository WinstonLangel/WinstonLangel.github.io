**标注要点：**

- 标注文件会自动生成同名的 `.json` 文件并放在同样的文件夹中

### 三、数据转换 Data converting

YOLO需要的是 `.txt` 格式的标注文件，需要把labelme的json转成YOLO格式。

#### 转换代码demo `convert.py`：
```python
import json
from pathlib import Path

# 14个商标的映射（汉字 → 数字）
class_map = {
    '麦当劳': 0,
    '霸王茶姬': 1,
    # ... 其他12个商标
}

images_dir = Path('images')
labels_dir = Path('labels')
labels_dir.mkdir(exist_ok=True)

for json_file in images_dir.glob('*.json'):
    with open(json_file, 'r', encoding='utf-8') as f:
        data = json.load(f)
    
    img_w = data['imageWidth']
    img_h = data['imageHeight']
    
    txt_file = labels_dir / (json_file.stem + '.txt')
    
    with open(txt_file, 'w') as f:
        for shape in data['shapes']:
            label = shape['label']
            class_id = class_map[label]
            
            # 计算YOLO格式的坐标
            points = shape['points']
            x_coords = [p[0] for p in points]
            y_coords = [p[1] for p in points]
            x_min, x_max = min(x_coords), max(x_coords)
            y_min, y_max = min(y_coords), max(y_coords)
            
            x_center = (x_min + x_max) / 2 / img_w
            y_center = (y_min + y_max) / 2 / img_h
            width = (x_max - x_min) / img_w
            height = (y_max - y_min) / img_h
            
            f.write(f"{class_id} {x_center:.6f} {y_center:.6f} {width:.6f} {height:.6f}\n")
```

### 四、配置文件demo示例

#### `data.yaml`
```yaml
path: D:/python-learn/python-YOLOv8
train: images
val: images
nc: 14  # 14个商标
names: ['麦当劳', '霸王茶姬', '肯德基', ...]  # 14个名字
```

**Warnings：**
- `nc` = number of classes（类别数）
- `names` 的顺序要和 `class_map` 里的数字对应

### 五、开始训练 Now let's begin!!

#### `train.py`demo
```python
from ultralytics import YOLO

model = YOLO('yolov8n.pt')  # 加载预训练模型

results = model.train(
    data='data.yaml',
    epochs=100,      # 训练100轮
    imgsz=640,       # 图片大小
    batch=8,         # 每次处理8张
    device='cpu',    # 用CPU训练
    name='MYmodel'  # 模型名字
)

print("✅ 训练完成！")
```

**训练过程中的发现：**
- 第一次运行会下载字体文件（`Arial.Unicode.ttf`）
- 下载速度很慢（27KB/s），因为服务器在国外

### 六、测试模型

#### `test.py`
```python
from ultralytics import YOLO

model = YOLO('runs/detect/14pinpai/weights/best.pt')

# 14个商标名字
brands = ['麦当劳', '霸王茶姬', '肯德基', ...]

results = model('images/test.jpg')

for result in results:
    result.show()  # 显示图片
    
    boxes = result.boxes
    if len(boxes) > 0:
        for box in boxes:
            class_id = int(box.cls[0])
            conf = box.conf[0]
            print(f"🎉 找到 {brands[class_id]}，置信度: {conf:.2f}")
    else:
        print("😢 没找到商标")
```

## 📚 学到的知识点 What I have learned...

### 1. YOLO的核心概念 Core concepts
- **标注**：告诉AI"这就是你要找的东西"
- **训练**：让AI看很多照片学会识别
- **模型**：训练完生成的文件（`.pt`）
- **推理**：用模型识别新照片

### 2. 重要参数 Key parameters
| 参数 | 作用 | 我的理解 |
|------|------|---------|
| `epochs` | 训练轮数 | 越多学得越好，但可能太久 |
| `batch` | 一次看几张 | 8-16比较合适 |
| `imgsz` | 图片大小 | 640是标准尺寸 |
| `device` | 用CPU还是GPU | 没GPU就用cpu |

### 3. 遇到的坑 Some problems
- ❌ 下载慢（换国内镜像解决）
- ❌ 不知道labelme怎么用（使用快捷键更快）
- ❌ 汉字标签怎么处理（直接用，YOLO支持）
